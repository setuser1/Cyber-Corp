If you watched all the way to the end of the new Apple TV show “Pluribus,” you may have noticed an unusual disclaimer in the credits: “This show was made by humans.”

That terse message —  placed right below a note that “animal wranglers were on set to ensure animal safety” — might provide a model for other filmmakers seeking to highlight that their work was made without the use of generative AI.
