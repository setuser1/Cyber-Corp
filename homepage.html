Subnetting

Subnetting is the way to divide IP networks into smaller, subnetworks, they use CIDR notation or full 4 octet notation.

  • CIDR Notation: /24 equals 255.255.255.0
  • Class A Networks: 10.0.0.0/8
  • Class B Networks: 172.16.0.0/12
  • Class C Networks: 192.168.0.0/16
  • Default Subnet Mask: Changes based on network class
  • Private IP Ranges: Reserved for internal networks
